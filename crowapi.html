Crow Fact API

The StopError Crow API enables programmatic access to veritable Crow Facts, easliy allowing software developers to integrate Crow Facts into their applications and deliver them to end-users.


How to use

Simply make a HTTP request to "crowfact.stoperror.dev/api/get-fact". You will recieve a JSON response containing a crow fact.


Windows Example

PS> Invoke-RestMethod crowfact.stoperror.dev/api/get-fact


macOS/Linux Example

$ curl crowfact.stoperror.dev/api/get-fact
